# Features

This document lists features planned for future versions of the project.

---

- Support for text **above media**
- Accurate handling of messages marked as **edited**

- Stable captions in **document albums**, including cases where each document has its own caption
- Improved handling of long messages, including better placement of timestamps
  and sender names without forced message splitting
- Support for additional content types:
  - locations
  - contacts
  - stories

---
